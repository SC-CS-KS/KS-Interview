# 229. Majority Element II - 多数元素 2 -

给定大小为n的整数数组，找到所有出现超过⌊n /3⌋次的元素。
注意：算法需要时间复杂度为 线性 和 空间复杂度为 O(1)。

Example 1:

Input: [3,2,3] Output: [3] Example 2:

Input: [1,1,1,3,3,2,2,2] Output: [1,2]
